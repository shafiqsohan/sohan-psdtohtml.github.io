Folder Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "traffic" folder.
You will find the index.html, about.html, blog.html, blog-single.html, case-study-details.html, case-study-v1.html, case-study-v2.html, contact-v1.html, contact-v2.html, faq.html, home-2.html, home-3.html, home-4.html, pricing.html, qoute.html, service-details.html, service-v1.html, service-v2.html, team.html, 404.html, file on the root that will be edited.
Also you will see 4 directories, css, js, fonts, img and images.
In the first folder Content you will find style.css and other styling files the all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts.
